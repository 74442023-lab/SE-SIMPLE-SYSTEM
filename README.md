# SE-SIMPLE-SYSTEM

A simple console-based task management system created for learning basic system development and version control using Git and GitHub.

##  System Description
This project is a basic menu-driven task manager where users can:
- Add new tasks  
- View all existing tasks  
- Exit the program  

It demonstrates essential programming concepts such as:
- Loops  
- Conditional statements  
- Functions  
- User input handling
